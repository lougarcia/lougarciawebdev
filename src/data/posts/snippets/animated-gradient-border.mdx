# Animated Gradient Border

Using registered custom properties

## Setup

### HTML

```html
<button>Click me</button>
```

### CSS
```css
@property --deg {
    syntax: "<angle>";
    initial-value: 0turn;
    inherits: true;
}

@keyframes rotate {
    to {
        --deg: 1turn;
    }
}

button {
    border: 4px solid transparent;
    background:
        linear-gradient(white) padding-box,
        conic-gradient(
            from var(--deg),
            limegreen,
            yellow,
            limegreen
        ) border-box;
    animation: rotate 2s linear infinite;

    /* These will depend on your theme */
    font: inherit;
    color: black;
    padding: .25em 1em;
    border-radius: .25rem;
}
```

## Rotate On-Hover only
Update the codes with the following to rotate on-hover only.

### CSS
```css
button {
    ...
    animation-play-state: paused;
}

button:hover,
button:focus-visible {
    animation-play-state: running;
}
```
