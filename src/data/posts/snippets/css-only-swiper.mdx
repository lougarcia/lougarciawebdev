# CSS-Only Swiper

A custom swiper using only CSS

## Setup

### HTML
```html
<div class="wrapper">
    <div class="item">1</div>
    <div class="item">2</div>
    <div class="item">3</div>
</div>
```

### CSS

```css
.wrapper {
    --window-width: 300px;
    display: flex;
    gap: 20px;
    width: var(--window-width);
    overflow-x: scroll;
    scroll-snap-type: x mandatory;
}
.item {
    scroll-snap-align: center;
    flex-shrink: 0;
    width: var(--window-width);

    /* These will depend on your theme */
    border-radius: 1rem;
    text-align: center;
    align-content: center;
    height: 300px;
    background-color: hsl(0 1 1);
    color: #ccc;
}
```

#### For vertical scrolling

Update the axis-related properties and use height instead of width.

```css
.wrapper {
    ...
    overflow-y: scroll;
    scroll-snap: type: y mandatory;
}
```
